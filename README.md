# NGUEM'S YAFE - Site de Présentation

Site ultra-rapide pour le salon Yafe au Palais des Congrès.

## Stack Technique

- **Astro** - Framework pour performance maximale
- **Tailwind CSS** - Design sur-mesure Mobile-First
- **Framer Motion** - Animations fluides
- **React** - Composants interactifs
- **Lucide React** - Icônes légères

## Démarrage Rapide

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Build pour production
npm run build

# Prévisualiser le build
npm run preview
```

## Déploiement

Le site est configuré pour être déployé sur Vercel :
- URL : nguems-yafe.vercel.app

## Structure

- `/src/pages` - Page principale
- `/src/components` - Composants React réutilisables
- `/src/layouts` - Layouts Astro
- `/public` - Assets statiques
